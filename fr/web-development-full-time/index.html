Qui est admissible?

Notre Bootcamp FullStack à temps plein est pour ceux qui cherchent une carrière en développement web. En 8 semaines, vous irez de débutant à développeur web avec l'aide de nos professeurs.

Pas sûr si ce cours est pour vous?
Contactez-nous et nous allons vous diriger dans la bonne direction.

Ce que vous allez apprendre

Afin de vous apprêter au marché du travail en 8 semaines, vous apprendrez les bases de la programmation, développement front-end et le développement back-end.

  • HTML5

    La formation de HTML qui vous rend compétent dans la création du code sémantique.

  • CSS3 and SASS

    Obtenez une base solide en CSS et la connaissance des possibilités de CSS3.

  • JavaScript

    Le langage rrincipale pour ajouter de l'interactivité à vos pages web.

  • SQL

    Apprenez à modéliser, stocker et interroger les données de votre site web.

  • APIs

    En savoir plus sur les API et ajouter des fonctionnalités à votre site.

  • NodeJS

    Apprendre un langage back-end populaire pour créer des sites Web dynamiques.

  • Foundation

    Apprenez Foundation, une librairie de web front-end.

  • jQuery

    Apprenez à créer des fonctions dynamiques pour vos sites Web.

  • React

    Utilisez une librairie UI populaire pour créer des applications d'une seule page.

  • Git and GitHub

    Apprenez à gérer les versions de fichiers, et le code en collaboration avec votre équipe.

  • Terminal

    Apprenez à exécuter des tâches journalières en utilisant la ligne de commande de votre ordinateur.

  • Grunt

    Apprenez à créer des processus automatisés et accélérez votre développement web.

Programme d'études

Dans cette section, nous allons apprendre les bases de la programmation en construisant diverses applications en ligne de commande. Vous aurez la chance de construire une application météo en ligne de commande, un carnet d'adresses ainsi que quelques jeux de hasard.

  • Mise en place de notre IDE et environnement de développement
  • Apprendre les bases de la ligne de commande
  • Utilisation de Git et GitHub
  • JavaScript et JSON
  • Introduction a NodeJS et npm
  • Caractéristiques de ECMAScript 6
  • Développement Orienté Objet

Dans cette section, nous allons nous concentrer sur la modélisation des données et la persistance avec MySQL. Nous allons aussi pratiquer l'exposition de données en créant un projet API qui comprennent la mise à niveau de notre carnet d'adresses et jeux à utiliser une base de données.

  • Présentation des options de persistance des données
  • Discussion sur SQL vs NoSQL
  • Utilisation de MySQL pour modéliser les données et les sauvegarder
  • Utilisation d'un ORM pour MySQL avec NodeJS
  • Exposer les données et les fonctions grâce à un API REST
  • Déploiement d'un serveur d'API sur la plate-forme Heroku

Après l'acquisition des connaissances requises pour créer un API, nous approfondirerons lors de ce volet nos compétences front-end en mettant à jour nos projets antérieurs avec des outils tels que jQuery, Backbone et/ou ReactJS.

  • Survol du CSS et HTML sémantique
  • Découverte de CSS3 et HTML5
  • Introduction à SASS
  • Utilisation de Grunt pour automatiser le dévelopment
  • Utilisation de jQuery, AJAX et APIs externes
  • Création d'une application d'un page avec ReactJS

Dans cette section, vous et votre instructeur allez travailler ensemble pour déterminer des sujets avancés que vous souhaitez explorer. Vous pouvez décider d'aller plus en profondeur sur les choses que nous avons déjà vu, ou de découvrir une nouvelle librairie, langage ou méthodologie. L'objectif est de créer un projet final avec ce que vous avez appris au cours du bootcamp.

Certains des sujets qui peuvent être abordés:

  • Introduction au développement WordPress
  • Contribution à un projet open-source
  • Utilisation d'une librairie front-end différente comme Backbone ou AngularJS
  • Exploration d'un système de base de données différent, comme MongoDB ou Cassandra
  • Déploiement d'une application sur Amazon Web Services
  • Essayer le développement mobile en utilisant JavaScript avec React Native
  • Utilizer le CSS3 avancé pour crée un nouveau CSS Zen Garden (csszengarden.com)
  • ... ou toute autre chose qui pique votre curiosité et vous garde motivé :)

Le dernier jour du bootcamp, vous présenterez votre projet à vos camarades de classe, amis et famille, ainsi qu'à nos partenaires d'emploi.
