What you will learn

Our part-time curriculum is an excellent introduction into the world of web development. We focus on the fundamental languages of web development: HTML & CSS. We also cover some productivity tools such as GitHub, Zurb's Foundation Framework, and WebStorm. Upon completion of the course, you will be able to build landing pages or multi-page websites and put them on the web.

Curriculum

  • Overview of the schedule
  • Setup of development environment
  • Quick talk about HTML and CSS
  • Quick talk about Git and Github
  • Brief overview of how the web works (servers, clients, etc.)
  • Brief history of HTML
  • Introduction to HTML's role, structure vs. presentation
  • Understanding what is meant by HTML5
  • Using HTML semantic tags to create meaningful documents
  • Understanding of images, differences between the various image formats
  • Introduction to CSS' role, structure vs. presentation
  • Experimenting with basic CSS properties
  • Understanding of the CSS cascade and specificity
  • Understanding of the CSS box model
  • Use a browser's inspector to interact with the content
  • High-level discussion about responsiveness
  • Using our newly-acquired understanding of front-end technologies, explore the Foundation framework
  • Demo: creating a small landing page using Foundation’s responsive components

Using everything that was learned through the sessions, create a personal landing page or a microsite. This project can be done alone or with a group. Your instructor will control the scope of the project, and will be guiding you throughout.
