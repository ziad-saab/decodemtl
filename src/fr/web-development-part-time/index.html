Qui est admissible?

Notre introduction au développement web s'addresse à tous. Notre objectif est de vous enseigner des connaissances générales dans le domaine du web, en vous apportant l'expérience pratique nécessaire afin que vous puissiez construire, gérer et développer vos sites web.

Ce cours est idéal pour:

  • Les Entrepreneurs qui cherchent à développer un MVP
  • Les Designers qui veulent donner vie à leurs créations
  • Les Chef de produit qui souhaitent acquérir des compétences plus techniques
  • Les Salariés plein-temps qui ne peuvent pas (encore) quitter leur job
  • Les Passionnés de technologie prêts à contrôler le monde
  • Les Futurs étudiants plein-temps qui veulent mettre un pied dans l'eau avant de se jeter

Pas sûr si ce cours est pour vous?
Contactez-nous et nous allons vous diriger dans la bonne direction.

Ce que vous allez apprendre

Que vous lisiez un simple article ou utilisiez l'application web la plus complexe au monde, chaque page que vous voyez dans votre navigateur est bâtie avec HTML et CSS. Ce couple puissant est la fondation du développement web et le parfait point de départ d'une aventure de codeur.

Dans ce cours, vous apprendrez à écrire du code irréprochable, conforme aux standards du web. Vous vous familiariserez avec les concepts du design inhérents au web. Le premier jour, nous installerons votre environnement de travail avec un éditeur de texte. Et au bout de 10 semaines passées ensemble, vous aurez votre premier site web réactif en ligne.
